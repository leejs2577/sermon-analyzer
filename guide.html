5. 내보내기 기능

결과를 다양한 형식으로 저장

분석이 완료되면 결과 화면 상단에 내보내기 도구모음이 표시됩니다.

내보내기

Markdown (.md)

MD 버튼

Obsidian, Notion, Logseq 등 노트 앱에 바로 임포트할 수 있는 형식입니다. 날짜·설교자·본문 등 frontmatter 메타데이터가 포함됩니다.

이미지 (.png)

이미지 버튼

결과 화면 전체를 고해상도(2배) PNG 이미지로 캡처합니다. SNS 공유나 인쇄에 적합합니다. 다크모드가 적용된 상태로도 캡처됩니다.

독립형 HTML (.html)

HTML 버튼

모든 스타일이 내장된 독립 HTML 파일로 내려받습니다. 브라우저에서 바로 열 수 있으며, 이메일 첨부나 장기 보관에 적합합니다.

내려받은 파일은 브라우저 기본 다운로드 폴더에 저장됩니다. 파일명은 날짜_설교제목.확장자 형식입니다.
